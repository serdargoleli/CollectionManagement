## 📁 Core Klasörü Yapısı

```plaintext
core/
├── constants/       # Uygulama genelinde kullanılan sabit değerler
├── models/          # Veri modelleri (veri yapıları ve ilişkileri)
├── services/        # İş mantığı ve veri işleme servisleri
├── store/           # Global durum yönetimi (state management) 
└── types/           # Uygulama genelinde kullanılan TypeScript tip tanımları
```
